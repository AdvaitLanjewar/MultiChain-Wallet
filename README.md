# 🚀 MultiChain Wallet

A Full Stack Multi-Chain Cryptocurrency Wallet built using React.js, Node.js and Express.

## Features

- Generate Ethereum Wallet
- Generate Bitcoin Wallet
- Generate Solana Wallet
- Import Existing Wallet
- View Portfolio
- Send Crypto
- Receive Crypto
- Buy Crypto
- Transaction History
- QR Code Support
- Live Crypto Prices
- Responsive Dashboard

## Tech Stack

Frontend
- React.js
- React Router
- Axios
- Chart.js
- Framer Motion

Backend
- Node.js
- Express.js

Blockchain
- Ethereum
- Bitcoin
- Solana
